프로그래머스 1단계 : 없는 숫자 더하기

0부터 9까지의 숫자 중 일부가 들어있는 정수 배열 numbers가 매개변수로 주어집니다.
numbers에서 찾을 수 없는 0부터 9까지의 숫자를 모두 찾아 더한 수를 return 하도록 solution 함수를 완성해주세요.

제한 사항

1 ≤ numbers의 길이 ≤ 9
0 ≤ numbers의 모든 원소 ≤ 9
numbers의 모든 원소는 서로 다릅니다.

나의 문제 풀이

number의 갯수 만큼 순차적으로 더해줍니다. 그 다음 나온 값에서 45를 빼주면 기존에 존재하지 않는 값을에 더한 값이 나오겠죠?
그렇게해서 풀이를 했습니다. 

function solution(numbers) {
    var answer = 0;

    for(var i=0;i<numbers.length;i++) {
        answer += numbers[i];
    }

    answer = 45 - answer;

    return answer;
}

다른 사람 문제 풀이

reduce() 메서드를 사용하여 각 없는 값을 더허여 빼준 것 같습니다.
reduce() 메서드는 배열의 여러개의 값을 하나의 값으로 축소해준다. 

function solution(numbers) {
    return 45 - numbers.reduce((cur, acc) => cur + acc, 0);
}
