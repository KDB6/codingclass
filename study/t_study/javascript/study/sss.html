slice() / substring() / substr()

위 세 가지의 메서드는 문자열에서 원하는 값을 추출하여 문자열로 반환한다.

"문자열".slice(시작위치)
"문자열".slice(시작위치, 끝나는위치)
// 시작 위치의 값은 끝나는 위치 값보다 작아야 합니다.
// substring() 시작 값이 끝나는 값보다 클 경우 두 값을 바꿔서 처리(에러 방지)
"문자열".substr(시작위치)
"문자열".substr(시작위치, 길이)
