프로그래머스 1단계 : 핸드폰 번호 가리기

프로그래머스 모바일은 개인정보 보호를 위해 고지서를 보낼 때 고객들의 전화번호의 일부를 가립니다.
전화번호가 문자열 phone_number로 주어졌을 때, 전화번호의 뒷 4자리를 제외한 나머지 숫자를 전부 *으로 가린 문자열을 리턴하는 함수, solution을 완성해주세요.

제한 사항

phone_number는 길이 4 이상, 20이하인 문자열입니다.

나의 문제 풀이

우선 num 갯수의 뒷 자리를 빼고 바꾸는 거기에 -4로 빼줍니다.
그 음은 *를 사용하여 가려야하기 때문에 repeat를 사용하여 *를 반복 또한 뒷자리는 substring사용하여 추출합니다. 

function solution(num) {
    const aa = num.length - 4
    return "*".repeat(aa) + num.substring(aa)
}

다른 사람 문제 풀이

맨 위에 있던 코드 입니다. 해당 분은 replace를 사용하여 문자열을 변경한 것 같습니다. 

function hide_numbers(s) {
    return s.replace(/\d(?=\d{4})/g, "*");
  }
