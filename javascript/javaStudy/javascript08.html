  • Number.MAX_VALUE : 표현할 수 있는 최대 숫자 값
  • Number.MIN_VALUE : 표현할 수 있는 최소 숫자 값
  • Number.MAX_SAFE_INTEGER : 최대 안전 정수
  • Number.MIN_SAFE_INTEGER : 최소 안전 정수
  • Number.Nan : Not-a-Number 나온 값이 숫자가 아니라는 뜻이며, NaN일 경우 true 아닐 경우 false

  • Number.isInteger : 전달된 값이 정수인지 여부를 결정
  • Number.isNaN : 어떤 값이 NaN인지 판별
  • Number.prototype.toFixed : 소수 부분 자릿수를 전달받은 값으로 고정한 후, 그 값을 문자열로 반환
