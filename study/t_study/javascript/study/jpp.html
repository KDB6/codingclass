01. 배열 메서드 : join() : 배열 요소 문자열 결합 : 반환(문자열)

join() 메서드는 배열의 모든 요소 값을 문자열로 변경하며, 인수로 받은 문자로 연결해서 문장열로 반환한다.


    const arrNum = [100, 200, 300, 400, 500];
    const text1 = arrNum.join('');      // 기본값
    const text2 = arrNum.join(' ');     // 띄어쓰기
    const text3 = arrNum.join('*');     // *
    const text4 = arrNum.join('-');     // -
메서드 결과값
join('') 100200300400500
join(' ') 100 200 300 400 500
join('*') 100*200*300*400*500
join('-') 100-200-300-400-500

02. 배열 메서드 : push()

push() 메서드는 배열의 끝에 하나 이상의 요소를 추가 후 배열의 새로운 길이를 반환한다.


    const arrNum = [100, 200, 300, 400, 500]
    const arrPush = arrNum.push(600);

    document.querySelector(".sample01_M1").innerHTML = "push(600)";
    document.querySelector(".sample01_P1").innerHTML = arrPush;  // 6
    document.querySelector(".sample01_A1").innerHTML = arrNum;  // 100, 200 , 300, 400, 500, 600
메서드 리턴값 결과값
push(600) 6 100, 200, 300, 400, 500, 600

03. 배열 메서드 : pop()

pop() 메서드는 배열에서 마지막 요소를 제거하고 그 요소를 반환한다.


    const arrNum = [100, 200, 300, 400, 500]
    const arrPush = arrNum.pop();

    document.querySelector(".sample03_M1").innerHTML = "pop()";
    document.querySelector(".sample03_P1").innerHTML = arrPush;      // 500
    document.querySelector(".sample03_A1").innerHTML = arrNum;      // 100, 200 , 300, 400
메서드 리턴값 결과값
pop() 500 100, 200, 300, 400
