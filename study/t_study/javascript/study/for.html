반복문

반복문은 프로그램에에서 필요한 결과 값을 도출하기 위해 실행문의 순서를 반복적으로 실행시키는 문법을 말합니다. 반복문에는 while문, do while문, for문이 있으며, for문을 가장 많이 사용합니다.

for (초기 값 ; 조건식; 증감값) {
  실행문
}

while문

}

do while문

}

for문

for (초기 값 ; 조건식; 증감값) {
  실행문
}

for문을 이용해서 구구단 만들기

for문도 if문과 마찬가지로 for문안에 또 다른 for문을 중첩 사용 가능

    for(let i=2; i<=9; i++) {
        for(let j=1; j<=9; j++) {
            let sum = i * j;
            document.write(i + " * " + j + " = " + sum);
        }
    }

중첩 for문을 이용해서 테이블을 만들 수 있습니다.

    let table = "<table class='testTable'>";

    table += "<tr>"
    for (let i=1; i<=10; i++) {
        table += "";

            for (let j=1; j<=10; j++){

            table += "</tr>";
