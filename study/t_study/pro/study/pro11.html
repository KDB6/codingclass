프로그래머스 1단계 : 정수 내림차순으로 배치하기

함수 solution은 정수 n을 매개변수로 입력받습니다.
n의 각 자릿수를 큰것부터 작은 순으로 정렬한 새로운 정수를 리턴해주세요.
예를들어 n이 118372면 873211을 리턴하면 됩니다.

제한 사항

n은 1이상 8000000000 이하인 자연수입니다.

나의 문제 풀이

parseInt이용해 정수로 변환 후 split를 사용하여 원하는 값을 추출 그 후 sort사용하여 정렬해주고 reverse로 반대로 정렬 trim 공백제거후 배치해주기 

function solution(n) {
    return (parseInt(n.toString().split('').sort().reverse().join('').trim()))
}

다른 사람 문제 풀이

상수로 각각 정수를 지정해 준 뒤에, split 값을 추출 sort 정렬 reverse 반대로 정렬 그 후 join을 사용한 것 같습니다. 

function solution(n) {
    const newN = n + "";
    const newArr = newN
      .split("")
      .sort()
      .reverse()
      .join("");
  
    return +newArr;
}
