프로그래머스 1단계 : 문자열 내림차순으로 배치하기

문자열 s에 나타나는 문자를 큰것부터 작은 순으로 정렬해 새로운 문자열을 리턴하는 함수, solution을 완성해주세요.
s는 영문 대소문자로만 구성되어 있으며, 대문자는 소문자보다 작은 것으로 간주합니다.

제한 사항

str은 길이 1 이상인 문자열입니다.

나의 문제 풀이

내림차순으로 내리기 위하여 처음에는 sort를 사용하여 정렬을 해주고, 그 다음 reverse를 사용하여 다시 반대로 내림차순으로 변경
그 후에 join을 사용하여 문자열로 만들어 줬습니다. 

function solution(s) {
    var answer = '';
    var str=s.split("");

    return str.sort().reverse().join("");
}

다른 사람 문제 풀이

이 분은 위와 같은 개념으로 더 깔끔하게 했습니다. 

function solution(s) {
    return s
      .split("")
      .sort()
      .reverse()
      .join("");
}
