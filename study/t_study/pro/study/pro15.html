프로그래머스 1단계 : 서울에서 김서방 찾기(김씨가.. 한 둘도 아니고..)

String형 배열 seoul의 element중 "Kim"의 위치 x를 찾아, "김서방은 x에 있다"는 String을 반환하는 함수,solution을 완성하세요.
seoul에 "Kim"은 오직 한 번만 나타나며 잘못된 값이 입력되는 경우는 없습니다.

제한 사항

seoul은 길이 1 이상, 1000 이하인 배열입니다.
seoul의 원소는 길이 1 이상, 20 이하인 문자열입니다.
"Kim"은 반드시 seoul 안에 포함되어 있습니다.

나의 문제 풀이

변수에 indexOf를 사용하여 Kim을 지정해주고 answer를 출력해줍니다.
indexOf는 문자열에서 특정 문자의 위치를 찾는 메서드입니다. 

function solution(aa) {
    let bb = aa.indexOf("Kim");
    var answer = '김서방은 '+`${bb}`+'에 있다';
    return answer
}

다른 사람 문제 풀이

이 분 께서는 findIndex 메서드를 쓰셨습니다.
findIndex는 배열 요소를 검색할 때 쓰는 메서드입니다. 

const solution = (arr) => `김서방은 ${arr.findIndex(s => s === 'Kim')}에 있다`;
