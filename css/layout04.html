Layout

레이아웃은 한 공간에 여러 구성 요소를 효율적으로 배치하는 작업이다.
웹 페이지에서의 레이아웃은 디자인과 외관의 결정을 짓는 중요한 요소이다.

float

float속성은 복잡한 형태의 레이아웃을 구성하는데 필요한 핵심 속성으로써 떠있게, 흐르도록 하는 속성입니다.
해당 레이아웃은 float을 이용해서 구성하였습니다.

flex

flex속성은 구성 요소를 유연하고 동적이게 만드는 박스 모델입니다.

grid

grid속성은 페이지를 여러 주요 영역으로 나누거나, 크기와 위치 계층 구조로 나눌 때 적합한 역할입니다.
grid를 사용하여 레이아웃을 더 쉽게 구현할 수 있습니다.
