프로그래머스 1단계 : 하샤드 수

양의 정수 x가 하샤드 수이려면 x의 자릿수의 합으로 x가 나누어져야 합니다.
예를 들어 18의 자릿수 합은 1+8=9이고, 18은 9로 나누어 떨어지므로 18은 하샤드 수입니다.
자연수 x를 입력받아 x가 하샤드 수인지 아닌지 검사하는 함수, solution을 완성해주세요.

제한 사항

x는 1 이상, 10000 이하인 정수입니다.

나의 문제 풀이

split()과 reduce()를 이용하여 구했습니다. 

function solution(n){
    return n % (""+n).split("").reduce((a, b) => +a + +b) == 0;
}

다른 사람 문제 풀이

각 변수를 지정해주고 do while문을 사용하며 그 와중에 Math.floor로 소주점을 버렸습니다. 

function solution(x) {
    let num = x;
    let sum = 0;
    do {
        sum += x%10;
        x = Math.floor(x/10);
    } while (x>0);

    return !(num%sum);
}
