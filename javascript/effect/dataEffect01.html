1부터 100까지 출력하기

1.1 while문을 이용해 1부터 100까지 출력하기

    let i = 1;

    while(i <= 100){
        document.write(i);
        i++;
    }
코드 보기 123456789...100

1.2 for문을 이용해 1부터 100까지 출력하기

    for( let i=1; i<=100; i++){
        document.write(i);
    }
코드 보기 123456789...100

1.3 for문을 이용해 1부터 100까지 출력하기(짝수, 홀수) 

    // 짝수
    for( let i=2; i<=100; i+=2){
        document.write(i);
    }

    /홀수
    for( let i=1; i<=100; i+=2){
        document.write(i);
    }
코드 보기 짝수 246810...100
홀수 13579...99

1.4 for문을 이용해 1부터 100까지 합을 출력하기

    let sum = 0;

    for(let i=0; i<=100; i++) {
        sum += i;
    }
    document.write(sum)
코드 보기 5050
